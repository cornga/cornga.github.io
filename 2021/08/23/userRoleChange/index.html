SSHR-V2.0 自動維護角色

維護Boss角色

在progress 判斷是否是主管

在 MongoDB 維護主管角色

維護富港聘僱人員角色

富港聘僱人員 將角色由預設的 N_USER (3) 改為CUINT_USER (4)

Oracle中抓db_name & d_i_cd

 oracle api >>db_apis >>dbname.js
 this.l_dbName == "CUINT" ||this.l_dbName == "TSCI"  >>富港人員

login.vue 中比對

富港聘僱人員 將角色由預設的 N_USER (3) 改為CUINT_USER (4)
富港轉正崴  將角色由CUINT_USER 改為 N_USER
正崴直接人員 將角色由改為 D_USER >> id:5
富港直接人員 將角色由改為 D_CUINT_USER >> id:6

MongoDB 執行變更角色

mongo api >> controller >> user-router.js >> userRoleChange

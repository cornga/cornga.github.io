JustNote建置記錄

hexo –version

確認相關環境已建置 >> node ,git ,hexo

建立justNote專案

啟hexo server

Hexo 5 之後最大特色大概就是主題終於使用 npm 來安裝而不是使用 git clone，當然 Hexo 5 還是有保留原有的 theme/[主題名稱] 建立方式‧

透過 npm 來安裝 NexT

 
安裝最新版本： npm install hexo-theme-next@latest

Theme 改套用 >> next

複製config 檔，到根目錄，以後編輯此檔即可調整theme

cp node_modules/hexo-theme-next/_config.yml _config.next.yml

發佈到github

安裝 hexo-deployer-git
